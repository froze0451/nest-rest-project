## Description

NestJS + MongoDB REST-api example. 
Includes some validation, data transformation and custom exceptions. Installed MongoDB required

## Installation

```bash
$ npm install
#or
$ yarn
```

## Running the app

```bash
$ npm run start
#or
$ yarn run start
```

## Api
```bash
# HOST browse using Postman / Insomina or other
http://localhost:3000/
```
